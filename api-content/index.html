模板化却迥然不同的喜剧风格 这两部电影在塑造喜剧时用了完全不同的手法，如果要一决高下的话，我认为李焕英更好。唐探 3 的喜剧风格基本沿用唐探 2， 非常模板化，主要集中在唐仁这个角色上，具体手法无非就是唐仁的口音、各种变装、黄段子（带孩子或者父母去看需谨慎）、怀旧 BGM 等，倒不是说模板化完全不好，毕竟喜剧创作本身有一些通用技巧，只不过唐探 3 的喜剧表现方式较为传统，当然，也并非全程无亮点，一群人在医院转移尸体，最终在电梯里相遇的情景，兼顾紧张与搞笑，简直就是精简版《疯狂的石头》。 李焕英这部电影的模板化体现在强烈的小品风格上，负责体现这种风格的主要是沈腾饰演的沈光林这个角色，小品中常见的笑点是金句，沈腾在宣传片中就抛出了「她叫焕英，我叫光林，我俩欢迎光临，听着多热情」这个谐音金句，再配上沈腾极具特色的表情，让人梦回春晚，还有一个让我印象比较深刻的小笑点，就是沈腾在电影院准备换座位的一幕，我们的视线随着镜头看着沈腾在一排观众前穿梭，就要到座位前的时候，另一哥们突然一屁股坐上去了（好吧，我描述得很枯燥），类似的场景也经常在小品中出现。 各自的硬伤 糟糕的推理与俗套的叙事 从第一部至第三部，唐探系列的口碑呈直线下滑趋势，这其中的重要原因就是推理元素越来越弱，具体到唐探 3，因为观众与编剧的信息完全不对等，观看电影时就会产生以下情况：开场 5 分钟之内大概就可以猜到凶手，但是直到案情揭晓的最后时刻，观众都不可能拼凑出完整的事实，以上情况倒也可以理解，毕竟这种手法在侦探影视中并不少见，但是唐探 3 还存在另外一个更大的问题——其向观众隐藏起来的元素出现得极其不合理，比如托尼贾前往黑龙江绥化调查长泽雅美的身世，这成了解释作案动机的关键，但是为什么是绥化呢？最起码也要编个类似于「之前查到长泽雅美与绥化有某种关联」的剧情来铺垫一下吧，除此之外，车子里面的方巾也令人费解，简直在侮辱东京警察以及长泽雅美这个人物的智商，竟然放那么久都没人来拿。 唐探 3 推理的另外一个问题是喜欢糊弄观众，剧中多次提到侦探小说以及相关名词，如「本格」[1]「三口棺材」[2]「犹大之窗」[3]，看起来非常硬核，但是结合前面说到的信息不对等的问题可知，唐探 3 并非本格派推理，也没有对人性的深刻反思，所以也无法划入社会派[4]推理的范畴，所以当秦风在念叨着「本格」的时候，我只觉得陈思诚在故弄玄虚。另外，剧中还有一个较大的争议，即所谓的「第十四种密室」，部分推理迷认为陈思诚过于自大了，认为自己创造了一种全新的密室类型，我对推理了解不多，很难直接着手分析，查了一些资料，感觉有些复杂，有兴趣的可以自行探索。 除了推理之外，唐探 3 还有一个问题就是叙事俗套，主要体现在神秘组织 Q 上，一群高智商人士躲在世界暗处纵横捭阖，审判世界，这是哪个年代的俗套叙事啊！比如至今仍在互联网流传的「罗斯柴尔家族」，还有所谓「奶头乐」的来源，都是这种叙事的体现，作为成年人，感觉这种设定实在是太过中二了。不过我猜想陈思诚可能是出于无奈，从电影网剧的联动来看，他正在试图打造「唐探宇宙」，而这种宇宙必须借助团体来实现，比如漫威宇宙的复仇者联盟。 最后，唐探 3 还有一个看起来像是智商欠费的槽点，就是片中突然跳出的「和平精英五大灵童」，不多说了，史上最令人反胃的植入广告。 不够立体的母爱
